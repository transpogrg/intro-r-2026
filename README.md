# Intro to R 

Instructors: Tammy Lee, Lindsey Smith, Iris Rademacher  
Location: Engineering Building, Room 310, 1930 SW 4th Avenue, Portland, Oregon 97201  
Dates: 3/18 - 3/20, 9:30am - 4:30pm   

### Prerequisite
Basic knowledge and experience of conducting research with quantitative information; experience, or keen to learn, a programming language and data processing and statistical software.

### Format
Classes will all be hands-on sessions with some lecture, discussion, and mostly practical. There will be some assigned readings and the expectation is to read them and participate in class discussions. Highlights of the class include creating a reproducible work product in which participants go through the process of retreiving data, processing, and analyzing data while learning best practices of data science.
